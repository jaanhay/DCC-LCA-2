# Bully Election Algorithm

## Aim
To implement Bully Election Algorithm in C.

## Description
The Bully Algorithm is used to elect a coordinator in a distributed system.
The process with the highest ID becomes the coordinator.

## Steps
1. Initialize processes
2. Simulate failure of a process
3. Start election from a lower process
4. Higher ID processes respond
5. Highest active process becomes coordinator
